# MS-102 Study Notes / Apuntes de estudio MS-102

Personal study notes for the **Microsoft MS-102: Microsoft 365 Administrator** certification exam.

These notes are available in two languages:

- 🇬🇧 [MS-102 study notes - English](./MS-102-study-notes.en.md)
- 🇪🇸 [Notas de estudio MS-102 - Español](./MS-102-study-notes.es.md)

The notes are organized around the main areas represented in my revision material:

- Deploy and manage a Microsoft 365 tenant

- Manage security and threats with Microsoft Defender XDR
